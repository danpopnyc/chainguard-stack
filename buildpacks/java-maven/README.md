# Sample: Java Maven Buildpack

Compatible apps:
- Java apps that use Maven

### Usage

```bash
pack build sample-java-maven-app --builder chainguard-builder:wolfi --buildpack . --path ../../apps/java-maven
```

OR

```bash
pack build sample-java-maven-app --builder chainguard-builder:wolfi --buildpack . --path ../../apps/java-maven
```
